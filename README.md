这里是BUAA **2023年秋季**的**计算机组成原理课**(**实验部分**) 的内容，此仓库将进行**所有课下历次作业**的源码分享🎁

## 简要指导🔔
**`.circ`后缀文件用`logisim`软件打开，是参考的电路图文件**    

**`.v`后缀文件用`ISE`软件打开，是`ISE`软件的主要`Verilog`文件**  

**`.asm`后缀文件用`Mars`软件打开，是`MIPS`汇编程序文件**  （部分`.asm`文件有相应同名的`.c`文件帮助理解）
